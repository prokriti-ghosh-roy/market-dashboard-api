# Market Dashboard Tool
A Python-based CLI tool that fetches real-time stock and cryptocurrency data using the Yahoo Finance API.

## Features
* **Real-time Data:** Fetches live prices and currency information.
* **Interactive Interface:** Allows users to query multiple stocks in one session.
* **Error Handling:** Validates ticker symbols to prevent program crashes.

## Tech Stack
* **Language:** Python 3.x
* **Libraries:** `yfinance`

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install yfinance`
3. Run the script: `python main.py`
